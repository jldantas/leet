# LEET
Leverage EDR for Execution of Things

## Getting started

### Prerequisites

```
Python >= 3.6
```

For the CLI:

```
tabulate
```

For the Carbon Black backend:

```
apscheduler
cbapi
```

### Installation

```
pip install tabulate apscheduler cbapi
```

Clone the git repository and run: `python leet.py`
